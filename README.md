# SDU-Math-rwteefz-2025
This repository collects past exams, review notes, and selected study materials for the 2025 Dual Bachelor's Degree Program in Finance and Mathematics  at Shandong University.  The goal is to make course review easier for future students by keeping useful materials in one structured place.
## Courses

* Mathematical Analysis
* Advanced Algebra
* Point-Set Topology
* Complex Analysis
* Real Analysis
* Abstract Algebra
* Ordinary Differential Equations
* Differential Geometry
* Numerical Analysis
* Probability and Mathematical Statistics

## Repository Scope

This repository is intended for:
* past exam papers and recalled exam questions
* review notes and summaries (e.g., Euler-Maclaurin formula expansions)
* selected problem sets or solutions useful for exam preparation
* course-specific reference notes created or collected for study

This repository is **not** intended to archive full copyrighted textbooks, private personal submissions, account credentials, or temporary build files.

## Notes

* Some materials may be incomplete, unofficial, or based on student recollection.
* Please use the materials only for study and review.
* If any file infringes copyright, contains private information, or should not be public, please open an issue or contact the maintainer so it can be removed.

## Contributing

Contributions are welcome. Please keep the repository focused on review-friendly materials and avoid uploading:
* full commercial textbooks or unauthorized scans
* files containing personal information
* password files, tokens, or other credentials
* LaTeX build artifacts and other generated temporary files

When adding materials, place them under the relevant course folder and use clear file names whenever possible.
